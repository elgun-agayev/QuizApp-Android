# 🚀 Quiz App Android

A modern and feature-rich Quiz Application built with **Kotlin**, designed to support both **single-player** and **multiplayer game modes**, including real-time interactions and competitive gameplay.

This project demonstrates advanced Android development concepts such as **modular architecture**, **state management**, and **scalable UI design**.

---

## ✨ Features

### 🎮 Gameplay

* Single Player quiz mode with customizable settings
* Online Duel mode (1v1 matchmaking concept)
* Multiplayer Team Battle (Team A vs Team B)
* Challenge system between players

### 👥 Social & Interaction

* Add friends system
* Send and receive game requests
* Challenge other players
* Player profile viewing

### 🏆 Competitive System

* Leaderboard (weekly / monthly / overall)
* Score tracking
* Top performer detection

### 💬 Real-time Features (Simulated / Structured)

* Room-based multiplayer logic
* Team balancing system
* In-game chat structure

### ⚙️ User System

* User profile management
* Avatar selection
* Theme switching (Light / Dark / System)

### 🎯 Gameplay Utilities

* 50/50 helper
* Skip question
* Extra time feature

## 🏗️ Architecture Overview

The project follows a **clean and scalable MVVM-based structure**:

* `ui/` → Fragments & UI logic
* `adapter/` → RecyclerView adapters
* `model/` → Data models
* `manager/` → Business logic & state handling

This separation ensures:

* Maintainability
* Scalability
* Clear responsibility layers
